# Emma OCC v2.0

A mobile-first PWA foundation for your personal operations control center.

## What it includes
- React + Vite app
- PWA manifest
- App icon
- Roster snapshot
- NS roster code reference
- Commute timeline
- Risk simulator
- Email action center placeholder
- Weather placeholder
- Integration control panel

## Install locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
```

## Deploy
Recommended:
- Vercel
- Netlify
- GitHub Pages

After deployment, open the HTTPS URL in Safari and use:
Share → Add to Home Screen.
